# Отклонения и незапланированные изменения

Документ фиксирует расхождения с ТЗ v1.0 и дополнительные решения, которые были сделаны по ходу работ. Цель — сохранить прозрачность и управляемость объёма проекта.

## 1. Сводка соответствия ТЗ

В целом направление верное: стек соответствует (Next.js, TS, Tailwind, R3F, tsparticles, Framer Motion, Supabase, Upstash), структура проекта совпадает с ТЗ, ключевые фичи Hero + форма + API + wall присутствуют. Есть ряд дополнений и несколько пунктов ТЗ, которые пока не реализованы полностью.

## 2. Дополнения, не описанные в ТЗ (предлагается оставить)

| Изменение | Причина | Влияние | Рекомендация |
|---|---|---|---|
| Фоновая карусель фото через `data/background-carousel.json` + API `app/api/admin/background-carousel/route.ts` | Требование загрузки фото через admin-консоль, но без UI — минимально жизнеспособный механизм | Упрощает управление фоном, можно заменить UI позже | Оставить, дописать UI позже |
| Проксирование фото из локальной папки через `app/api/photo/[...path]` и rewrite `/photo/:path*` | Для локальной разработки и VPS без публичного CDN | Упрощает деплой, но требует аккуратной настройки `PHOTO_ROOT` | Оставить, добавить предупреждение по безопасности |
| Fallback для 3D «50» при отсутствии WebGL | Совместимость с низкими девайсами/браузерами | Улучшает доступность | Оставить |
| Оптимизация производительности 3D (DPR cap, frameloop "demand") | Снижение нагрузки на мобильных и слабых устройствах | Улучшает стабильность | Оставить |
| Бикон для увеличения просмотров `POST /api/congratulations/[slug]/view` | Исключение лишнего server render и сохранение счетчика | Безопаснее и корректнее для статистики | Оставить |
| CI workflow с env secrets в e2e job | Нужен запуск E2E с Supabase | Улучшает стабильность CI | Оставить |
| `.gitattributes` (LF) | Предотвращение CRLF конфликтов в CI | Стабилизирует линт/формат | Оставить |

## 3. Расхождения с ТЗ (нужны решения)

| Пункт ТЗ | Текущее состояние | Что решить |
|---|---|---|
| Next.js 15 | Сейчас `next: ^16.2.1` | Либо подтвердить апгрейд, либо даунгрейд до 15 |

| Статичный режим на всех мобильных | Сейчас только `(max-width: 360px)` | Расширить до всех mobile (userAgent/width < 1024/ pointer: coarse) |
| Admin console для карусели | Пока только JSON + API | Нужен UI (минимум: загрузка/список/удаление) |
